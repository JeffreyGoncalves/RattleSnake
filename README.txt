# RattleSnake

la ligne de commande pour éxécuter notre programme est :

/opt/anaconda3/bin/python main.py  [mode] [police]

[mode] : mode du bot que l'on souhaite utiliser ( 1 pour le backchannel bot, 2 pour psychiatrist bot, 3 pour notre bot amélioré).

[police] : choix de police pour la conversation avec le chatbot (0 pour la police basique et 1 pour une police vaporwave).
